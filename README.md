# Bonus Tent

**Bonus Tent** is a cozy, atmospheric Minecraft Fabric mod developed by **David Lite Studio** for Minecraft Java **1.16.5**.

This mod replaces the vanilla starting bonus chest with an abandoned explorer's tent that serves as the perfect start for your survival adventure.

## Features

- **Explorer's Camp Spawning**: Replaces the default vanilla bonus chest with a fully generated cozy explorer's tent complete with a campfire, survival chest, and utility barrel.
- **Smart Flat Surface Finder**: Uses an advanced chunk scanner to find a flat 5x5 clearing of grass or dirt near the world spawn point, preventing the tent from spawning inside tree canopies or floating in the air.
- **Visual Smoke Beacon**: Spawns a tall column of flame and campfire smoke particles rising 30-40 blocks high above the campfire when the world generates, serving as a beacon so players never lose their starting camp.
- **Thematic Survival Chest Loot**:
  - Guaranteed stone axe and stone pickaxe, randomized with 20% to 60% wear (damage).
  - 100% guaranteed Flint and Steel, heavily worn down to exactly **5 uses left** (damage set to 0.921875).
  - Starting food supply (apples, bread), sticks, oak logs, and torches.
  - A **35% chance** to find a jack-pot drop of 1 to 3 diamonds.
- **Thematic Barrel Supplies**: The barrel contains starting utility materials like sticks (10-15), bones, string, gravel, and dirt.
- **Lawn Care Alignment**: Cleans up exposed dirt directly under the tent and its perimeter, replacing it with grass blocks for a natural, seamless integration into the environment.

## Requirements

- **Minecraft**: `1.16.5`
- **Fabric Loader**: `>=0.14.0`
- **Fabric API** (Compatible version for 1.16.5)
